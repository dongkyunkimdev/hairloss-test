# Hairloss-test - 탈모 테스트 웹  

Hairloss-test는 Google Teachable machine을 사용한 머신러닝 기반 탈모 테스트 사이트입니다.  
정수리 사진을 업로드하여 결과를 확인할 수 있습니다.  
**표본이 충분하지 않아 결과의 신뢰도가 떨어질 수 있습니다.**

## 사용 기술

언어
* HTML5
* CSS3
* JQUERY

운영체제
* Windows 10

서버
* Netlify

머신러닝 도구
* Teachable machine

## 주요 기능

* 정수리 사진 업로드하여 탈모 테스트
